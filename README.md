## Usage

### Install uv

```bash
cd path/to/root/directory
pip install uv
```

### Option A: Web API Server

```bash
uv run fastapi dev main.py
```

Access web app at `http://127.0.0.1:8000` or `http://localhost:8000`.

### Option B: CLI

```bash
uv run cli.py
```
